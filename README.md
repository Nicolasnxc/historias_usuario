# CaféTech - Sistema de Pedidos

**Descripción:** Aplicación diseñada para optimizar la gestión y venta de productos de una cafetería. Permite a los clientes explorar el catálogo de bebidas, gestionar su carrito de compras y realizar pedidos de manera rápida y segura.

## Funcionalidades Principales (Flujo Guiado)
1. **Registro de usuario:** crear cuenta con email y contraseña.
2. **Inicio de sesión:** acceso seguro con validación.
3. **Explorar menú:** ver lista de cafés, precios y descripciones.
4. **Agregar productos al carrito:** seleccionar cantidad y productos.
5. **Realizar pedido:** confirmación, método de pago simulado y número de orden.
6. **Historial de pedido:** consultar órdenes anteriores.
